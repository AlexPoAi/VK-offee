## ✅ Рекомендация

**Добавить в автоматическую синхронизацию:**

Создать скрипт `sync-google-sheets.sh`:

```bash
#!/bin/bash
# Синхронизация Google Sheets → CSV

SHEETS=(
    "1Ai4qUIXpkc1fJ_Cd5Vh-wgPV9a-blOhtQWV_tyqhFTU:372408211:Калькуляции СЕБЕСТОИМОСТЬ КУХНЯ"
)

for sheet in "${SHEETS[@]}"; do
    IFS=':' read -r sheet_id gid name <<< "$sheet"

    echo "📥 Скачивание: $name"
    curl -L "https://docs.google.com/spreadsheets/d/${sheet_id}/export?format=csv&gid=${gid}" \
         -o "/Users/alexander/VK-offee/knowledge-base/КУХНЯ/${name}.csv"

    echo "✅ Сохранено: ${name}.csv"
done
```

**Добавить в cron:**
```bash
# После синхронизации Google Drive
5 9 * * * /Users/alexander/VK-offee/.github/scripts/sync-google-sheets.sh
```
