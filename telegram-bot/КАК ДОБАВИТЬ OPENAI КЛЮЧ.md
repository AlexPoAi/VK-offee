# 🔑 Как добавить OpenAI API ключ для Telegram-бота

**Дата:** 2026-02-19
**Статус:** ⚠️ Нужно добавить API ключ

---

## 🎯 Зачем нужен OpenAI API ключ?

Telegram-бот использует **GPT-4o-mini** для:
- AI-ответов на вопросы о кофейне
- Поиска информации в базе знаний
- Умных рекомендаций

**Без ключа:** Бот будет работать, но без AI-ответов (только поиск файлов)
**С ключом:** Бот будет давать умные ответы на основе базы знаний

---

## 📝 Как получить OpenAI API ключ

### Шаг 1: Зарегистрироваться на OpenAI

1. Перейти на https://platform.openai.com/
2. Нажать "Sign up" или "Log in"
3. Зарегистрироваться через email или Google

### Шаг 2: Создать API ключ

1. Перейти в раздел "API keys": https://platform.openai.com/api-keys
2. Нажать "Create new secret key"
3. Дать название ключу (например: "VK-offee Telegram Bot")
4. Скопировать ключ (он показывается только один раз!)

**Ключ выглядит так:**
```
sk-proj-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

### Шаг 3: Добавить баланс

1. Перейти в "Billing": https://platform.openai.com/settings/organization/billing/overview
2. Нажать "Add payment method"
3. Добавить карту
4. Пополнить баланс (минимум $5)

**Стоимость GPT-4o-mini:**
- Входящие токены: $0.15 / 1M токенов
- Исходящие токены: $0.60 / 1M токенов
- **Примерно:** $0.01-0.05 за 100 сообщений

## 🔧 Как добавить ключ в бота

### Вариант 1: Через файл .env (локально)

```bash
# Открыть файл
nano /Users/alexander/VK-offee/telegram-bot/.env

# Заменить строку:
OPENAI_API_KEY=your_openai_api_key_here

# На реальный ключ:
OPENAI_API_KEY=sk-proj-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

# Сохранить: Ctrl+O, Enter, Ctrl+X
```

### Вариант 2: Через переменные окружения

```bash
# Добавить в ~/.zshrc или ~/.bashrc
export OPENAI_API_KEY="sk-proj-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"

# Применить
source ~/.zshrc
```

### Вариант 3: Через Railway (если бот в облаке)

1. Открыть проект на Railway
2. Перейти в "Variables"
3. Добавить переменную:
   - Name: `OPENAI_API_KEY`
   - Value: `sk-proj-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`
4. Сохранить и перезапустить бот

## 💰 Стоимость использования

### GPT-4o-mini (рекомендуется):

**Пример расчёта:**
- 1000 сообщений в месяц
- Средний запрос: 500 токенов вход + 200 токенов выход
- Стоимость: ~$0.50-1.00 в месяц

**Очень дёшево!** 💰

### Альтернативы:

1. **GPT-3.5-turbo** — ещё дешевле, но менее умный
2. **GPT-4** — дороже, но умнее
3. **Локальная модель** — бесплатно, но нужен мощный сервер

## 🔒 Безопасность

### ⚠️ ВАЖНО:

1. **Не публикуй ключ в GitHub!**
   - Файл `.env` уже в `.gitignore`
   - Проверь: `git status` не должен показывать `.env`

2. **Не делись ключом!**
   - Это как пароль от карты
   - Если кто-то узнает → может потратить твои деньги

3. **Ограничь расходы:**
   - В OpenAI Dashboard → "Usage limits"
   - Установи лимит: $10/месяц

4. **Мониторь использование:**
   - Проверяй раз в неделю: https://platform.openai.com/usage

## 📞 Если что-то не работает

### Проблема 1: "Invalid API key"

**Причина:** Неправильный ключ или не активирован

**Решение:**
1. Проверь ключ в .env (нет лишних пробелов)
2. Проверь, что ключ активен на OpenAI
3. Проверь, что есть баланс

### Проблема 2: "Rate limit exceeded"

**Причина:** Превышен лимит запросов

**Решение:**
1. Подожди 1 минуту
2. Увеличь лимит в OpenAI Dashboard
3. Пополни баланс

### Проблема 3: "Insufficient quota"

**Причина:** Закончились деньги на балансе

**Решение:**
1. Пополни баланс на OpenAI
2. Проверь, что карта активна
