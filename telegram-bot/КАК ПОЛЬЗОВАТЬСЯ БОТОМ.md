## 📱 Что умеет бот?

### 1. Отвечать на вопросы о кофейнях

**Примеры:**
- "Как приготовить капучино?"
- "Какие поставщики кофе?"
- "Сколько стоит латте?"
- "Какой рецепт раф-кофе?"

### 2. Искать в базе знаний

**Команда:** `/search <запрос>`

**Примеры:**
```
/search бариста
/search меню
/search поставщики
/search налоги
```

### 3. Показывать роли FPF (F1-F9)

**Команда:** `/roles`

Покажет таблицу 3×3 с описанием всех ролей.

### 4. Проверять статус

**Команда:** `/status`

Покажет, работает ли бот и доступна ли база знаний.

## 🚀 Как запустить бота?

### Проблема: Бот сейчас не работает

Последний раз работал 14 февраля на сервере Railway.

### Решение 1: Запустить локально (на Mac)

```bash
cd ~/VK-offee/telegram-bot
source venv/bin/activate
python3 bot.py
```

**Но:** Будет конфликт, если бот уже запущен на Railway.

### Решение 2: Перезапустить на Railway

1. Зайди на [railway.app](https://railway.app)
2. Найди проект VK-offee
3. Нажми "Restart"
4. Проверь логи

### Решение 3: Попросить меня запустить

Скажи: "Запусти Telegram-бота локально"

Я:
1. Проверю, не запущен ли он уже
2. Запущу бота в фоно��ом режиме
3. Проверю, что он работает

## 🔑 Нужен ли API ключ?

**Да, для умных ответов нужен OpenAI API ключ.**

Если его нет, бот будет работать, но только показывать результаты поиска без AI-анализа.

Проверить:
```bash
cat ~/VK-offee/telegram-bot/.env
```

Если нет `OPENAI_API_KEY`, скажи мне, я помогу добавить.

## ❓ Частые вопросы

### Q: Где найти бота в Telegram?

**A:** Открой @BotFather в Telegram, отправь `/mybots`, найди своего бота.

### Q: Бот не отвечает?

**A:** Проверь, запущен ли он:
```bash
ps aux | grep bot.py
```

Если нет, запусти:
```bash
cd ~/VK-offee/telegram-bot && ./start.sh
```

### Q: Бот отвечает "Информация не найдена"?

**A:** Возможно, база знаний не синхронизирована. Проверь:
```bash
ls ~/VK-offee/knowledge-base/
```

### Q: Можно ли добавить новые команды?

**A:** Да! Скажи, какие команды нужны, я добавлю в bot.py.
