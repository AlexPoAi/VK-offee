# HR.WP.001 — ГПХ Бариста

## Определение

Договор гражданско-правового характера для бариста кофейни «Вкусный кофе». Регулирует условия найма, стажировки, оплаты и расторжения договора.

## Критерии существования

Документ существует когда:
- Подписан обеими сторонами (Исполнитель + Заказчик)
- Заполнены паспортные данные Исполнителя
- Указаны даты начала и окончания

## Ключевые условия

| Параметр | Значение |
|----------|---------|
| Стажировка | до 14 рабочих дней |
| Оплата стажировки | 1500–2500 руб/смена |
| Первый день | ознакомительный, неоплачиваемый |
| Выплата стажировки | только при полном прохождении + сдаче экзамена |
| Отработка при увольнении | 14 календарных дней |
| Выплата при увольнении | после инвентаризации и вычета недостачи |

## Условия окончания стажировки

- Изучить составы продукции холодильной витрины
- Изучить калькуляции напитков
- Изучить ДИ Помощника бариста
- Сдать экзамен Шеф-бариста
