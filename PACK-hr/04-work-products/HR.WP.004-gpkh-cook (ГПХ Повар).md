# HR.WP.004 — ГПХ Повар

## Определение

Договор гражданско-правового характера для повара кофейни «Вкусный кофе». Регулирует условия найма, стажировки, оплаты и расторжения договора.

## Критерии существования

Документ существует когда:
- Подписан обеими сторонами (Исполнитель + Заказчик)
- Заполнены паспортные данные Исполнителя
- Указаны даты начала и окончания

## Ключевые условия

| Параметр | Значение |
|----------|---------|
| Стажировка | до 14 рабочих дней |
| Оплата стажировки | 1500–2500 руб/смена |
| Первый день | ознакомительный, неоплачиваемый |
| Выплата стажировки | только при полном прохождении + сдаче экзамена Шеф-повара |
| Отработка при увольнении | 14 календарных дней |
| Выплата при увольнении | после инвентаризации и вычета недостачи |

## Условия окончания стажировки

- Изучить технологические карты блюд и заготовок
- Изучить систему цветовой маркировки инвентаря и принцип FIFO
- Изучить стандарты гигиены и безопасности питания
- Изучить ДИ Повара
- Сдать экзамен Шеф-повара
