# Объектная модель PACK-hr

## Объекты

### [тип] сотрудник
```
DOMAIN: PACK-hr
ТИП: сотрудник

1. ОПРЕДЕЛЕНИЕ
   1 человек, выполняющий работу в кофейной точке

2. СОСТАВ
   нет

3. СВЯЗИ
   - кофейная точка  → [кофейная точка]
   - смена           → [смена]
   - период          → [период учёта]
   - договор         → [договор]
   - роли            → [роль]

4. ХАРАКТЕРИСТИКИ
   - имя
   - дата начала работы

5. СОСТОЯНИЯ
   - активен
   - уволен

6. ПРАВИЛА
   - сотрудник может иметь несколько ролей
   - сотрудник может участвовать в нескольких сменах
   - «бариста / кассир / официант» = роль, не отдельный тип (A.7)

7. ПРИМЕР
   Иван, бариста, работает с 01.01.2026
```
