# HR.WP.003 — ГПХ Раннер

## Определение

Договор гражданско-правового характера для раннера кофейни «Вкусный кофе». Применяется в том числе для несовершеннолетних (содержит блок согласия законного представителя).

## Критерии существования

Документ существует когда:
- Подписан обеими сторонами (Исполнитель + Заказчик)
- Для несовершеннолетних: подписан законным представителем
- Заполнены паспортные данные Исполнителя и место оказания услуг

## Ключевые условия

| Параметр | Значение |
|----------|---------|
| Ставка | 180 руб/час |
| Стажировка | не менее 30 рабочих часов |
| Оплата стажировки | 150 руб/час |
| Первый день | ознакомительный, неоплачиваемый |
| Выплата стажировки | только при полном прохождении и выходе в полные смены |
| Оплата | 1 раз в неделю (Пн-Вт) |

## Предмет договора (что делает раннер)

**Входит в обязанности:**
- Поддержание чистоты: уборка рабочих поверхностей, столов, витрин; вынос мусора; мытьё посуды
- Работа с продукцией: разогрев готовых блюд в микроволновой печи / на гриле
- Фасовка и упаковка продуктов для выдачи клиентам
- Расстановка товаров на витринах
- Покупка продуктов и расходных материалов (по согласованию)
- Перемещение товаров между кофейнями (в черте города)

**Не входит в обязанности:**
- Приготовление кофе (работа с кофемашиной)
- Обслуживание клиентов за кассой

## Особые условия для несовершеннолетних

- Запрет работы в ночное время (22:00–06:00)
- Максимальная продолжительность: до 12 часов в день
- Запрет опасных видов деятельности
- Требуется согласие законного представителя
