# HR.WP.002 — ГПХ Официант

## Определение

Договор гражданско-правового характера для официанта кофейни «Вкусный кофе». Регулирует условия найма, стажировки, оплаты и расторжения договора.

## Критерии существования

Документ существует когда:
- Подписан обеими сторонами (Исполнитель + Заказчик)
- Заполнены паспортные данные Исполнителя
- Указаны даты начала и окончания

## Ключевые условия

| Параметр | Значение |
|----------|---------|
| Стажировка | до 7 рабочих дней |
| Оплата стажировки | 100 руб/час, смена не менее 8 часов |
| Первый день | ознакомительный, неоплачиваемый |
| Выплата стажировки | только при полном прохождении + сдаче экзаменов Шеф-бариста |
| Отработка при увольнении | 14 календарных дней |
| Выплата при увольнении | после инвентаризации и вычета недостачи |

## Условия окончания стажировки

- Изучить составы продукции холодильной витрины (десерты, еда)
- Изучить меню напитков (самостоятельная рекомендация гостю)
- Изучить ДИ Официанта
- Сдать экзамен Шеф-бариста
