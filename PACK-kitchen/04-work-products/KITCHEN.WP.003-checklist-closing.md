# KITCHEN.WP.003 (Чек-лист закрытия смены)

## Определение

Чек-лист действий повара при закрытии смены для завершения работы и подготовки кухни к следующей смене.

## Содержание

### Производственные задачи

1. Приготовить заготовки "На несколько смен" (если требуется)
2. Промаркировать все заготовки (наименование, дата, вес)
3. Разместить заготовки по принципу FIFO
4. Внести данные в тетрадь учёта

### Уборка (детальный чек-лист)

**Зона кухни:**
- [ ] Помыть посуду, расставить на стеллаже для сушки
- [ ] Помыть раковину (антижир + моющее средство)
- [ ] Влажная уборка рабочих поверхностей (навесные полки, фартук, столы, холодильник, нижние полки, место под духовкой, стена у мусорного бака, фольга у плиты)
- [ ] Помыть духовку внутри и снаружи (снять и помыть решетки)
- [ ] Обработать кухонные тряпки (хозяйственное мыло + антижир)
- [ ] Помыть коврик в душевой кабине, оставить сушить
- [ ] Подмести и помыть пол (белизна + доместос + антижир)
- [ ] Собрать и выкинуть мусор

**Зона балкона:**
- [ ] Протереть дверцу холодильника, входную дверь, крышку морозильной камеры (кубика), верх морозильной камеры (вертикальной)
- [ ] Подмести и помыть пол

**Зона мытья посуды (коридор):**
- [ ] Стеллаж для сушки обработать антижиром, протереть насухо
- [ ] Помыть посуду, расставить на стеллаже
- [ ] Обработать фартук кафельный и раковину
- [ ] Помыть коврик в душевой кабине, оставить сушить
- [ ] Подмести и помыть пол до конца коридора

**Зона персонального туалета (для Раннера):**
- [ ] Помыть раковину и унитаз
- [ ] Помыть зеркало, протереть от разводов
- [ ] Помыть душевую кабину: дверцы, кран, поддон
- [ ] Протереть стены с моющим средством
- [ ] Помыть полы

### Административные задачи

5. Создать заявку на продукты (Пн, Ср, Пт)
6. Передать информацию следующей смене (если есть важные замечания)
