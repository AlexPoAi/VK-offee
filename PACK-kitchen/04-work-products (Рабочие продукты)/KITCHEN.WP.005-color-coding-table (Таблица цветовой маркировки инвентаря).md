# KITCHEN.WP.005 (Таблица цветовой маркировки инвентаря)

## Определение

Система цветовой маркировки инвентаря для предотвращения перекрестного загрязнения на кухне.

## Назначение

Обеспечить раздельное использование инвентаря для разных типов продуктов и предотвратить контаминацию.

## Цветовая схема

| Цвет | Назначение | Инвентарь |
|------|-----------|-----------|
| **Красный** | Сырое мясо | Доски, ножи, контейнеры |
| **Синий** | Сырая рыба | Доски, ножи, контейнеры |
| **Желтый** | Сырая птица | Доски, ножи, контейнеры |
| **Зеленый** | Овощи и фрукты (сырые) | Доски, ножи, контейнеры |
| **Белый** | Готовые продукты, хлеб, молочные | Доски, ножи, контейнеры |
| **Коричневый** | Овощи с кожурой (картофель, морковь) | Доски, ножи |

## Правила использования

1. **Строго соблюдать цветовую маркировку** — не использовать инвентарь одного цвета для других продуктов
2. **После использования** — немедленно мыть и дезинфицировать
3. **Хранение** — раздельно по цветам, в чистом виде
4. **Контроль** — ежедневная проверка состояния инвентаря
