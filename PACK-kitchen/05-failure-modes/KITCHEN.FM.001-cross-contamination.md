# KITCHEN.FM.001 (Перекрестное загрязнение)

## Определение

Типовая ошибка: использование одного инвентаря для разных типов продуктов без промежуточной обработки, что приводит к контаминации и риску пищевого отравления.

## Проявления

1. **Использование одной доски** для сырого мяса и готовых продуктов
2. **Использование одного ножа** для разных категорий продуктов без мытья
3. **Игнорирование цветовой маркировки** инвентаря
4. **Недостаточная дезинфекция** после работы с сырым мясом
5. **Использование одних тряпок** для разных поверхностей

## Последствия

- Пищевое отравление гостей
- Репутационный ущерб
- Штрафы от санитарных служб
- Закрытие заведения

## Профилактика

1. **Строго соблюдать цветовую маркировку** инвентаря (см. KITCHEN.WP.005)
2. **Немедленная дезинфекция** после работы с сырым мясом (протокол в KITCHEN.METHOD.005)
3. **Раздельное хранение** инвентаря по цветам
4. **Обучение персонала** правилам работы с инвентарем
5. **Ежедневный контроль** соблюдения правил

## Обнаружение

- Визуальный контроль использования инвентаря
- Проверка состояния досок и ножей
- Контроль процесса мытья и дезинфекции
