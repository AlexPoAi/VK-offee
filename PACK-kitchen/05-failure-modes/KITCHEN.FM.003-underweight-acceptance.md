# KITCHEN.FM.003 (Недовес при приемке товара)

## Определение

Типовая ошибка: принятие товара без взвешивания или с доверием к весу на упаковке, что приводит к недовесу и финансовым потерям.

## Проявления

1. **Доверие весу на упаковке** без проверки
2. **Пропуск взвешивания** из-за спешки или лени
3. **Подпись накладной ДО проверки** количества
4. **Игнорирование расхождений** "это же мелочь"
5. **Отсутствие весов** или неисправные весы при приемке

## Последствия

- Прямые финансовые потери (недополучение товара)
- Накопительный эффект (систематический недовес)
- Нехватка продуктов для выполнения плана
- Конфликты с поставщиками при позднем обнаружении
- Невозможность доказать недовес после подписания накладной

## Профилактика

1. **ОБЯЗАТЕЛЬНОЕ взвешивание/пересчет** ВСЕХ позиций
2. **Особое внимание** фасованному мясу/рыбе (вес нетто)
3. **Подпись накладной ТОЛЬКО после** полной проверки
4. **Немедленная фиксация** расхождений в накладной и Акте
5. **Связь с поставщиком** при обнаружении недовеса
6. **Исправные весы** всегда под рукой

## Обнаружение

- Взвешивание каждой позиции при приемке
- Сверка с накладной
- Контроль веса нетто (без упаковки) для фасованных продуктов
- Проверка количества штучных товаров

## Типовой пример

**Ситуация:** Поставщик привез 5 упаковок куриной грудки по 2 кг (по упаковке)
**Ошибка:** Повар принял товар, доверяя весу на упаковке, подписал накладную
**Результат:** При взвешивании на кухне обнаружен недовес 0,8 кг (реальный вес 9,2 кг вместо 10 кг)
**Последствия:** Невозможно доказать недовес, финансовые потери

**Правильно:**
1. Взвесить каждую упаковку при приемке
2. Обнаружить недовес ДО подписания
3. Зафиксировать в накладной и Акте расхождений
4. Связаться с поставщиком для решения
