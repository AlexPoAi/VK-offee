# KITCHEN.FM.002 (Нарушение FIFO)

## Определение

Типовая ошибка: нарушение принципа FIFO (First In, First Out) при размещении и использовании продуктов и заготовок, что приводит к порче продуктов и финансовым потерям.

## Проявления

1. **Размещение свежих заготовок впереди/сверху** старых
2. **Использование свежих продуктов** вместо более ранних
3. **Отсутствие маркировки** с датой приготовления
4. **Игнорирование дат** на стикерах при выборе заготовок
5. **Хаотичное размещение** продуктов без системы

## Последствия

- Порча продуктов с истекшим сроком годности
- Финансовые потери (списание испорченных продуктов)
- Риск использования просроченных продуктов
- Увеличение объема списаний

## Профилактика

1. **Обязательная маркировка** всех заготовок (наименование, дата, вес)
2. **Правильное размещение:**
   - Свежие заготовки → назад/вниз
   - Старые заготовки → вперед/вверх
3. **Контроль при использовании:** брать заготовки с более ранним сроком
4. **Ежедневная проверка** сроков годности
5. **Обучение персонала** принципу FIFO

## Обнаружение

- Проверка дат на стикерах заготовок
- Контроль порядка размещения в холодильниках
- Анализ объемов списаний
- Визуальный контроль при открытии смены
