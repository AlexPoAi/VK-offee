# KITCHEN.METHOD.004 (Приготовление заготовок)

## Определение

Метод приготовления и маркировки заготовок для блюд согласно техкартам с соблюдением принципа FIFO.

## Входы

- Техкарты заготовок
- Сырье (продукты)
- Инвентарь (раздельный по цветам)

## Выходы

- Приготовленные заготовки
- Промаркированные контейнеры (наименование, дата, вес)
- Записи в тетради учёта

## Ответственный

Повар-Универсал, Повар-Заготовщик

## Участники

- Раннер (мойка посуды, уборка)

## Процесс

### Общие принципы

1. Всегда использовать раздельный инвентарь (цветовая маркировка)
2. Приготовление строго по техкартам (раздел "Заготовки")
3. После приготовления:
   - Наклеить стикеры (наименование, дата, вес)
   - Внести данные в тетрадь
   - Разместить по принципу FIFO (свежие назад/вниз, старые вперед/вверх)
4. При использовании: брать заготовки с более ранним сроком приготовления

### Категории заготовок

**Заготовки "День в день"**
Готовятся в заявленном объёме на текущую смену, используются в эту же смену:
- Крупы: Киноа, Птитим, Рис
- Морепродукты: Заготовка креветок (маринад), Маринованная семга (слабосоленая)
- Для салатов: Начосы для боула с индейкой

**Заготовки "На несколько смен"**
Готовятся в конце рабочей смены до уборки кухни:
- Маринованное мясо птицы: курица, индейка
- Для салатов: Сухарики для салата Цезарь (готовим когда закончились, из целого батона)
- Соусы: Масаго, Соевый, Спайси (для боула с семгой), Чесночный (майонез с чесноком), Соус Шеф, Карри, Сметанный (для боула с индейкой), Томатный, Цезарь, Рисовый уксус
