# F1: Suprasystem / Предприниматель

**Контекст и рынок**

---

## Назначение

F1 отвечает на вопрос: **"В каком контексте существует VK-offee?"**

Описывает:
- Рынок HoReCa (кафе, рестораны, общепит)
- Конкурентов и тренды
- Целевую аудиторию (клиенты кофеен)
- Позиционирование на рынке
