# F2: Suprasystem / Инженер

**Окружение и интерфейсы**

---

## Назначение

F2 отвечает на вопрос: **"Как устроено окружение VK-offee?"**

Описывает:
- Интерфейсы с клиентами (меню, заказ, оплата)
- Точки контакта (витрина, касса, доставка)
- Физическое окружение (локация, интерьер)
- Цифровые интерфейсы (сайт, приложение, соцсети)

---

## Типы документов

| Тип | Описание | Примеры |
|-----|----------|---------|
| **Меню** | Что видит клиент | menu-design.md |
| **Точки контакта** | Где клиент взаимодействует | touchpoints.md |
| **Интерфейсы** | Как клиент заказывает | ordering-interfaces.md |
| **Локация** | Требования к расположению | location-requirements.md |
