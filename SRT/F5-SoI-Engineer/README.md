# F5: System-of-Interest / Инженер

**Архитектура продукта**

---

## Назначение

F5 отвечает на вопрос: **"Как устроена кофейня VK-offee?"**

Описывает:
- Архитектуру кофейни (зоны, оборудование)
- Меню и ассортимент (структура)
- Роли сотрудников
- Технологические карты

---

## Типы документов

| Тип | Описание | Примеры |
|-----|----------|---------|
| **Архитектура** | Планировка и зоны | cafe-layout.md |
| **Оборудование** | Список и требования | equipment-list.md |
| **Меню** | Структура ассортимента | menu-structure.md |
| **Роли** | Должности и функции | roles.md |
