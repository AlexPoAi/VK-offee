# F3: Suprasystem / Менеджер

**Взаимодействие и партнёрства**

---

## Назначение

F3 отвечает на вопрос: **"Как VK-offee взаимодействует с окружением?"**

Описывает:
- Партнёров (поставщики, франчайзи)
- Процессы взаимодействия с клиентами
- Маркетинговые активности
- Программы лояльности
