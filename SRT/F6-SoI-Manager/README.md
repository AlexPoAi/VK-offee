## Назначение

F6 отвечает на вопрос: **"Как работает кофейня VK-offee?"**

Описывает:
- Операционные процессы (открытие/закрытие смены)
- Рецепты и технологии приготовления
- Стандарты обслуживания
- Чек-листы и инструкции
