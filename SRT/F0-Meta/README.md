# F0: Метасемейство

**Управление хранилищем знаний**

---

## Назначение

F0 содержит документы об управлении самим хранилищем знаний VK-offee:
- Логика и онтология (как организованы знания)
- Отчёты и аналитика (что происходит с базой знаний)
- Inbox (входящие заметки для обработки)

---

## Типы документов

| Тип | Описание | Примеры |
|-----|----------|---------|
| **F0.1 - Логика** | Структура, онтология, правила | CLAUDE.md, INTEGRATION-PLAN.md |
| **F0.4 - Отчёты** | Автоматические отчёты, аналитика | strategy-session.md, audit-reports/ |
| **F0.9 - Inbox** | Входящие заметки для обработки | inbox/, temp-notes/ |

---

## Связь с другими семействами

F0 не описывает домен кофеен, а описывает саму систему управления знаниями.
