## 🧠 Ключевые инсайты

### Про синхронизацию Google Sheets:
- **Урок:** Прямые ссылки `/export?format=csv` не работают без авторизации
- **Правило:** Всегда использовать Google Sheets API с токеном
- **Применение:** Проверить все CSV файлы на наличие HTML

### Про Telegram-бота:
- **Урок:** Бот находит файлы по названию, но не читает содержимое
- **Проблема:** Логика поиска пропускает чтение файла после нахождения в названии
- **Исправление:** Убрана строка `continue` после нахождения в названии
