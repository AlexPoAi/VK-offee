```bash
# Установить cron задачу

# Результат:
# 0 9 * * * /Users/alexander/VK-offee/.github/scripts/auto-sync.sh >> /Users/alexander/VK-offee/sync-auto.log 2>&1
```

**Что произойдёт:**
- Каждый день в 9:00 запустится синхронизация
- Изменения автоматически закоммитятся в Git
- Отправятся в GitHub
- Логи сохранятся в `sync-auto.log`

### Вариант 2: Запускать вручную

```bash
cd /Users/alexander/VK-offee/.github/scripts
./auto-sync.sh
```

**Когда использовать:**
- Нужна срочная синхронизация
- Проверка работы скрипта
- Тестирование изменений

### Вариант 3: Через GitHub Actions (облако)

Создать `.github/workflows/sync-drive.yml`:

```yaml
name: Google Drive Sync

on:
  schedule:
    - cron: '0 9 * * *'  # Каждый день в 9:00 UTC
  workflow_dispatch:      # Ручной запуск

jobs:
  sync:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Set up Python
        uses: actions/setup-python@v4
        with:
          python-version: '3.11'
      - name: Install dependencies
        run: |
          cd .github/scripts
          pip install -r requirements.txt
      - name: Sync Google Drive
        env:
          GOOGLE_CREDENTIALS: ${{ secrets.GOOGLE_CREDENTIALS }}
        run: |
          cd .github/scripts
          python3 sync_google_drive_v2.py
      - name: Commit changes
        run: |
          git config --local user.email "action@github.com"
          git config --local user.name "GitHub Action"
          git add knowledge-base/
          git commit -m "Sync: Google Drive $(date +'%Y-%m-%d')" || exit 0
          git push
```

## 🔧 Устранение проблем

### Проблема 1: Синхронизация не запускается

**Причина:** cron не установлен или не работает

**Решение:**
```bash

crontab -l

# Установить заново
crontab /tmp/crontab_new

# Проверить права
chmod +x /Users/alexander/VK-offee/.github/scripts/auto-sync.sh
```

### Проблема 2: Ошибка авторизации Google

**Причина:** token.pickle устарел

**Решение:**
```bash
cd /Users/alexander/VK-offee/.github/scripts
rm token.pickle
python3 sync_google_drive_v2.py
# Пройти авторизацию заново
```

### Проблема 3: Конфликт Git

**Причина:** Локальные изменения не закоммичены

**Решение:**
```bash
cd /Users/alexander/VK-offee
git status
git add .
git commit -m "Manual commit before sync"
git push
```

### Проблема 4: Нет изменений, но коммиты создаются

**Причина:** Скрипт коммитит даже без изменений

**Решение:** Уже исправлено в `auto-sync.sh`:
```bash

    # Коммитить только если есть изменения
fi
```
