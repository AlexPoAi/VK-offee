## 🎯 Что такое VK-offee?

**VK-offee** — операционная база знаний сети кофеен «Вкусный Кофе».

**Цель:** Единый источник истины (Single Source of Truth) для всех операционных процессов.

**Архитектура:** FPF + SPF + SRT + FMT

---

## 📐 Четыре столпа архитектуры

```
FPF (First Principles Framework)
    ↓ Операционная система для мышления
SPF (Second Principles Framework)
    ↓ Методология создания Pack (доменного знания)
SRT (Systems-Roles-Table)
    ↓ Организация знаний через F0-F9
FMT (FMT-exocortex-template)
    ↓ Практические протоколы работы
```

## 🚦 Три режима работы

### 1. Режим чтения (Read Mode)
**Задача:** Найти информацию, ответить на вопрос.

**Алгоритм:**
1. Определи тему вопроса
2. Найди соответствующее семейство SRT (F0-F9)
3. Проверь PACK-cafe-operations/
4. Проверь knowledge-base/
5. Дай ответ со ссылками на источники

**Пример:** "Как готовить капучино?"
- → SRT/F6-SoI-Manager/ (процессы)
- → PACK-cafe-operations/03-methods/CAFE.METHOD.002-cappuccino.md
- → knowledge-base/БАР/Рецепты/

### 2. Режим создания (Create Mode)
**Задача:** Создать новый документ, метод, процесс.

**Алгоритм:**
1. Определи стадию процесса (process/00-11)
2. Используй шаблон из PACK-cafe-operations/_template/
3. Примени различения из distinctions.md
4. Определи систему и роль (F0-F9)
5. Запусти process-lint перед коммитом

**Пример:** "Создать метод приготовления латте"
- → process/07-method-and-product-extraction.md
- → PACK-cafe-operations/_template/03-methods/_method-card-template.md
- → Создать CAFE.METHOD.003-latte.md
- → Добавить в SRT/F6-SoI-Manager/

### 3. Режим анализа (Analysis Mode)
**Задача:** Проанализировать, найти проблемы, предложить улучшения.

**Алгоритм:**
1. Изучи bounded-context (границы домена)
2. Проверь distinctions (различения)
3. Найди failure-modes (типовые ошибки)
4. Проверь соответствие FPF принципам
5. Предложи улучшения

**Пример:** "Почему бот не находит информацию о зарплате?"
- → Проверь knowledge-base/ (есть ли файл?)
- → Проверь telegram-bot/bot.py (правильный ли поиск?)
- → Проверь PACK-cafe-operations/05-failure-modes/
- → Предложи решение
