# Документация для AI-агентов

Эта папка содержит универсальную документацию для всех AI-агентов (Claude, ChatGPT, Gemini, Llama и др.), работающих с репозиторием VK-offee.

---

## 📚 Файлы

| Файл | Назначение | Для кого |
|------|-----------|----------|
| **START-HERE.md** | Быстрый старт для агентов | Все агенты |
| **ARCHITECTURE.md** | Архитектура FPF+SPF+SRT+FMT | Все агенты |
| **RULES.md** | Правила работы с репозиторием | Все агенты |
| **PROCESS-LINT.md** | Проверка перед коммитом | Агенты, создающие контент |
| **EXAMPLES.md** | Примеры использования | Все агенты |
| **FAQ.md** | Частые вопросы | Все агенты |

---

## 🎯 Для разных платформ

### Claude Code
Читает `CLAUDE.md` в корне репозитория + эту папку.

### ChatGPT / GPT-4
Загрузите файлы из этой папки в контекст или используйте Custom Instructions.

### Google Gemini
Загрузите файлы из этой папки через Google AI Studio.

### Llama / Ollama
Используйте файлы как system prompt.

### Любой другой агент
Все файлы написаны в универсальном Markdown формате.

---

## 🚀 Быстрый старт

1. Прочитай **START-HERE.md**
2. Изучи **ARCHITECTURE.md**
3. Следуй **RULES.md**
4. При создании контента используй **PROCESS-LINT.md**
5. Смотри **EXAMPLES.md** для примеров
