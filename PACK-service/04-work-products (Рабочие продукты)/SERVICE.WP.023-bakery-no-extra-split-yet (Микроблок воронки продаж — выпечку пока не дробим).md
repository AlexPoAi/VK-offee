# Микроблок воронки продаж — выпечку пока не дробим

## Статус

Микроблок закрыт.

## Решение

Внутри ветки `выпечка` пока не создаём дополнительную развилку:

- не делим на `сладкая / сытная`
- не делим на отдельные подкатегории

На текущем этапе этого уровня детализации не нужно.

## Что должен сделать сотрудник

Если гость уже ушёл в ветку выпечки:

- не открывать новый уровень вопросов
- вести гостя дальше через конкретные позиции по ситуации

## Вывод

Ветка остаётся проще:

`еда -> выпечка -> конкретные предложения`
