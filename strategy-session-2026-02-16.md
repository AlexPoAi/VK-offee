## Диагностика

### 🟢 Что работает хорошо

**Операционные процессы (F6):**
- Чёткие стандарты обслуживания с метриками (P95, AOV, CRR, NPS)
- Детальные процессы: принять заказ → произвести → выдать → удержать
- Правила реакции на отклонения
- SLA между подсистемами (касса → бар → выдача ≤ 10 сек)

**Системное мышление:**
- Применение методов «чёрного» и «прозрачного» ящика
- Понимание холархии (надсистема → система → подсистемы)
- Анализ гостя как системы с внутренними процессами

**Ценности (F4):**
- Чёткие корпоративные ценности: порядочность, искренность, стрессоустойчивость, ответственность, доброжелательность
- Стратегия: "Делать то, что нравится, и быть в этом лучшими"

**Понимание надсистемы:**
- Детальный анализ районов (школы, офисы, ПВЗ, мечети, храмы)
- Понимание социальных маршрутов и концентрации трафика
- Учёт сезонности и пиков

### 🟡 Требует внимания

**Прибыльность (F7):**
- **КРИТИЧНО:** EBITDA 8-10% вместо целевых 15-20%
- Разрыв в 7-12 процентных пунктов = потеря 94,500-216,000₽/мес на точку
- При 3 точках: потеря 283,500-648,000₽/мес

**Отсутствие фактических данных:**
- Все документы содержат только целевые метрики
- Нет реальных данных по P95, AOV, CRR, NPS
- Невозможно оценить фактическое состояние бизнеса
- Нет данных о динамике (рост/падение)

**Устаревшая документация:**
- Последний пересмотр: декабрь 2025 (2 месяца назад)
- Следующий пересмотр: март 2026 (через месяц)
- Документы со статусом "yellow" требуют обновления

**Себестоимость напитков:**
- Данные есть, но статус "yellow"
- Нет информации о ценах продажи
- Невозможно рассчитать фактическую маржинальность
- Нет анализа самых/наименее прибыльных позиций

**Неполнота базы знаний:**
- Отсутствуют технологические карты напитков
- Нет рецептов приготовления всех позиций меню
- Нет стандартов качества обслуживания (детальных)
- Нет процедур открытия/закрытия смены (актуальных)

### 🔴 Критические проблемы

**1. Отсутствие системы сбора метрик**
- Нет инструментов для отслеживания P95, AOV, CRR, NPS
- Невозможно принимать решения на основе данных
- Нарушается принцип "метрики и данные" из F7

**2. Низкая прибыльность**
- EBITDA 8-10% критически ниже целевых 15-20%
- Причины не выявлены (нет данных для анализа)
- Возможные факторы:
  - Высокий фудкост (> 30%)
  - Неоптимальные зарплаты (> 30%)
  - Низкий средний чек (< 300₽)
  - Низкая конверсия трафика

**3. Отсутствие данных о команде (F9)**
- Нет информации об обучении
- Нет должностных инструкций
- Нет данных о текучести кадров
- Нет системы развития персонала

**4. Отсутствие данных о поставщиках (F8)**
- Нет информации о поставщиках
- Нет прайс-листов
- Нет анализа альтернативных поставщиков
- Риск зависимости от одного поставщика

**5. Нет конкурентного анализа (F1)**
- Упоминается кофейня "Чао" и другие конкуренты
- Нет детального анализа: цены, меню, качество, трафик
- Невозможно оценить конкурентное преимущество
