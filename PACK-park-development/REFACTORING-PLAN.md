# REFACTORING-PLAN — Полная реорганизация PACK-park-development

> План полного рефакторинга структуры проекта ПАРК для идеального входа агента за 5-10 минут.

**Дата создания:** 2026-04-15
**Статус:** in_progress
**Цель:** Агент, который никогда не видел проект, должен за 5-10 минут изучить его и дать точный ответ.

---

## Проблема (что было)

После встречи 15.04.2026 выявлено:
- Агент не может быстро войти в контекст проекта
- Даёт устаревшие рекомендации (как будто проект на 270 м², а не 324 м²)
- Не понимает текущий блокер (пакет АГО не получен)
- Не видит финансовую картину
- Теряет цепочку переговоров с ЛУКС

**Причина:** Несмотря на наличие всех карточек и документов, структура не оптимизирована для быстрого входа.

---

## Целевое состояние

### Критерии успеха

1. **Время входа:** 5-10 минут чтения
2. **Полнота понимания:** агент может ответить на 3 ключевых вопроса:
   - Где проект сейчас?
   - Что блокирует?
   - Что следующее?
3. **Все файлы имеют MD-карточки:** каждый PDF/DOCX/XLSX/JPG/PNG описан в MD
4. **Все файлы в одном месте:** нет зависимости от Downloads
5. **Чёткая структура дерева:** понятная иерархия папок и файлов

### Этап 2: Создание MD-карточек для всех файлов

**Цель:** Каждый PDF/DOCX/XLSX/JPG/PNG должен иметь MD-карточку с описанием содержимого.

- [ ] 2.1. Для каждого PDF создать MD-карточку:
  - Название документа
  - Дата создания/получения
  - Краткое описание содержимого (2-3 абзаца)
  - Ключевые данные (суммы, даты, стороны)
  - Связанные документы
  - Путь к оригиналу

- [ ] 2.2. Для каждого DOCX создать MD-карточку
- [ ] 2.3. Для каждого XLSX создать MD-карточку
- [ ] 2.4. Для каждого JPG/PNG создать MD-карточку
- [ ] 2.5. Для Telegram-экспорта создать MD-карточку с выжимкой

**Шаблон MD-карточки:**

```markdown
---
type: document
id: PARK.DOC.XXX
date: YYYY-MM-DD
document-type: тип-документа
source: путь/к/оригиналу
status: ok/draft/active
---

# PARK.DOC.XXX — Название документа

## Что это

Краткое описание (2-3 предложения).

## Ключевые данные

- Дата: ...
- Стороны: ...
- Сумма: ...
- Статус: ...

## Содержание

Подробное описание содержимого (2-3 абзаца).

## Связанные документы

- [PARK.DOC.YYY](...)
- [PARK.COMM.ZZZ](...)
```

**Артефакт:** MD-карточки для всех файлов
