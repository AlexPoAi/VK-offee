# PARK.DOC.028 — Приложения к заявлению на АГО для печати

```text
Приложения к заявлению
о согласовании архитектурно-градостроительного облика
объекта капитального строительства

1. Материалы по описанию архитектурно-градостроительного облика объекта капитального строительства, в том числе:
   1.1. пояснительная записка;
   1.2. фасадные решения;
   1.3. ведомость наружной отделки;
   1.4. 3D-визуализация;
   1.5. схема планировочной организации земельного участка / иные графические материалы комплекта.

2. Документ, подтверждающий полномочия представителя заявителя
   (при подаче заявления представителем).

3. Правоустанавливающие документы на земельный участок
   (при необходимости).

4. Иные документы по комплекту подачи
   (при наличии).
```

## Практическая задача листа

Этот лист удобно держать как:

- приложение к заявлению;
- внутренний чек-лист перед подачей;
- лист сверки пакета на месте в Бахчисарае.
