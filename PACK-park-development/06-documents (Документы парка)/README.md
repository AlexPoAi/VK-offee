# Документы парка Голубинка

Централизованное хранилище всех документов проекта парка.

## Структура

- **legal/** - Юридические документы (договоры, выписки, реквизиты)
- **technical/** - Технические документы (проекты, геология, ТЗ)
- **architectural/** - Архитектурные документы (планы, проекты, зонирование)
- **administrative/** - Административные документы (планы действий, переписка, регламенты)

## Синхронизация

Все документы синхронизированы с Google Drive (папка "Парк").
