# PARK.WP.017: Ответы на вопросы Константина по РНС

## Метаданные

**Дата подготовки:** 05.03.2026
**Тема:** Подготовка к рабочей встрече по вопросу получения разрешения на строительство (РНС)

## Описание

Документ содержит ответы на 5 ключевых вопросов Константина (архитектор) по проекту строительства кофейни в парке Голубинка:

1. На каком этапе выяснилась необходимость получения РНС?
2. Под какую категорию необходимо разрабатывать проектную документацию?
3. Какие разделы проектной документации необходимы для получения РНС?
4. Как категория здания влияет на стоимость строительства?
5. Есть ли понимание по лицензированию дальнейшей деятельности?

## Ключевые выводы

- Вопрос РНС поднят 13.11.2025
- Категория объекта: коммерческое общественно-деловое здание (ВРИ 4.9)
- Необходимые разделы: ПЗ, АР, КР, СПОЗУ, ПОС
- Приоритет: оптимизация стоимости первой очереди
- Лицензирование: после ввода в эксплуатацию, не зависит от конкретного оператора
