# PARK.COMM.030 — Вопрос по скамейкам и урнам для СПОЗУ

## Исходный запрос

```text
[27.04.2026 10:41] constantnine:
Александр, добрый день. Хотел бы уточнить для спозу - мы делаем в текущем проекте на благоустройстве территории какие-либо скамейки и урны?

[27.04.2026 18:15] ValerON:
Добрый день, очень ждем ответ.
Мы на финале по разделам уже
```

## Контекст

- В `PARK.DOC.010` планировка участка описывает здание, парковку, пешеходные дорожки, озеленение и инженерные коммуникации.
- В `PARK.DOC.023` СПОЗУ входит в пакет документов для РНС, но состав малых архитектурных форм отдельно не раскрыт.
- В старой переписке по концепции благоустройства обсуждалось, что не стоит обещать лишнее по благоустройству, чтобы это не стало обязательством первой очереди.
- Скамейки и урны как утверждённые элементы текущего проекта отдельно не зафиксированы.

## Позиция

В текущем проекте не стоит закладывать скамейки и урны как отдельный состав благоустройства.

Для СПОЗУ лучше оставить минимальный объём благоустройства:

- твёрдые покрытия / подходы;
- пешеходные связи;
- озеленение;
- парковку;
- инженерные привязки;
- без дополнительных малых архитектурных форм как обязательств первой очереди.

Допустимый компромисс: если проектировщикам для корректности раздела нужна санитарная / эксплуатационная точка, можно предусмотреть одну урну у входной группы. Скамейки не закладывать.

## Короткий ответ для отправки

```text
Добрый день. В текущем проекте скамейки и урны как отдельные элементы благоустройства не закладываем.

Для СПОЗУ показываем минимальное благоустройство территории: покрытия, подходы, пешеходные связи, озеленение, парковку и инженерные привязки. Скамейки не предусматриваем.

Если для корректности раздела нужна урна у входной группы, можно поставить одну урну у входа, но без расширения благоустройства и без дополнительных обязательств по первой очереди.
```
