# PARK.WP.039 — Чистовая карточка поездки на подачу АГО

## Назначение

Подготовить внешнюю версию карточки поездки без внутренней технички репозитория:

- без `type`, `id`, `work-product`, `PARK.WP.*` в теле документа;
- без служебных ссылок, не нужных в чтении с телефона или из Google Docs;
- в формате, который можно сразу открыть, прочитать и использовать в поездке.

## Что меняем

Из текущей карточки убираем:

- служебную мета-обвязку;
- внутренние обозначения рабочих продуктов;
- репозиторную терминологию, не несущую пользы в поездке.

Оставляем только:

- проверенные факты;
- маршрут;
- пакет документов;
- реплики для подачи;
- фиксацию результата.

## Критерий результата

На выходе должен быть один чистый документ, который:

1. не стыдно открыть перед третьим лицом;
2. понятен без знания внутренней структуры Pack;
3. годится для Google Docs и чтения с телефона.
