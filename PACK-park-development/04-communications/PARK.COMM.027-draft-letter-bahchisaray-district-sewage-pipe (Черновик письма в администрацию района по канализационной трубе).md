# PARK.COMM.027 — Письмо в администрацию района по канализационной трубе

## Назначение

Официальное обращение в Администрацию Бахчисарайского района по вопросу
балансодержателя, правового статуса и компетентного органа по канализационной трубе.

- документ materialized как `PARK.DOC.037`;
- есть живой Google Doc и отдельный PDF для печати;
- письмо доведено до одностраничного вида;
- пакет подачи облегчен до одного письма;
- копии и оригиналы подтверждающих документов готовы предоставить по запросу.

## Коммуникационный смысл

Цель этого письма не спорить с администрацией о трубе, а заставить контур района письменно назвать:
- балансодержателя;
- отсутствие сведений;
- либо точный компетентный адресат;
- либо выполнить письменную переадресацию по компетенции.
