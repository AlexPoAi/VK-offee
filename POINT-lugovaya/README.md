# POINT-lugovaya

## Назначение

`POINT-lugovaya` — point-level hub для точки `Луговая`.

Здесь живут локальные решения именно этой кофейни: контекст точки, статус локальных задач, дизайнерские концепты, ссылки на документы и все материалы, которые не стоит размазывать по общим каталогам без привязки к месту.

## Что хранится здесь

- `DESIGN/` — локальный дизайнерский поддомен точки

## Принцип

Общие стандарты сети, методы и переиспользуемые дизайнерские подходы остаются в `PACK-design`.

Все, что относится конкретно к Луговой как к уникальной точке, хранится в `POINT-lugovaya`.
