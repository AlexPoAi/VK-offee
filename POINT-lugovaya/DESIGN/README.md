# DESIGN subdomain for POINT-lugovaya

## Назначение

Этот поддомен собирает дизайнерские решения по точке `Луговая`.

## Что сюда складывать

- концепты фасада и входной группы
- решения по интерьеру
- локальные меню-носители и визуальные акценты
- референсы и варианты для подрядчиков
- финальные материалы по реализации

## Текущие материалы

- [Концепты](./concepts/)

## Связь с общим доменом

`PACK-design` отвечает за общую дизайнерскую систему сети:
- методы;
- термины;
- общие шаблоны;
- переиспользуемые решения.

`POINT-lugovaya/DESIGN` отвечает за локальную реализацию и уникальные решения именно для Луговой.
