# DESIGN.CONTRACT.001 — Границы домена PACK-design

## Назначение

`PACK-design` фиксирует визуальные решения сети `VK-offee` как отдельный рабочий домен, чтобы фасады, меню, вывески, интерьерные правки и бренд-материалы не терялись между чатами, референсами и разовыми файлами.

## Входы домена

- Фотографии текущего состояния точек
- Бизнес-задачи: повысить заметность, улучшить вход, увеличить продажи, обновить меню
- Референсы по стилю
- Ограничения по бюджету, срокам и материалам

## Выходы домена

- Концепты и мокапы
- Описания визуальных решений
- Наборы материалов для подрядчиков
- Визуальные носители для команды точки
- Библиотека решений, пригодных для повторного использования

## Основные объектные области

- Exterior: фасад, входная группа, вывеска, уличная навигация
- Interior: посадка, свет, акценты, фотозоны, мебельные решения
- Menu Design: настольное меню, тейблтенты, кассовые носители, digital board
- Brand Support: стилистические правила, референсы, кампании, визуальные подборки

## Критерии хорошего артефакта

1. Понятно, для какой точки и какого сценария он сделан.
2. Есть короткое описание цели и ожидаемого эффекта.
3. Артефакт можно показать подрядчику, управляющему или команде точки.
4. Его можно найти позже без восстановления контекста из памяти.
