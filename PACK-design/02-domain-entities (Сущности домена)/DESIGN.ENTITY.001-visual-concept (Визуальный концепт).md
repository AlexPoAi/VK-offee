# DESIGN.ENTITY.001 — Визуальный концепт

## Определение

Визуальный концепт — это оформленное предложение по изменению внешнего вида точки, зоны или носителя, которое соединяет эстетику, практичность и бизнес-цель.

## Атрибуты

- `location` — к какой точке относится
- `surface` — что именно меняем: фасад, вход, меню, интерьер, навигацию
- `goal` — какой эффект хотим получить
- `constraints` — бюджет, материалы, сроки, запреты
- `artifact_set` — изображения, тексты, мокапы, ссылки
- `status` — draft / review / approved / implemented

## Примеры

- Концепт входной группы Луговая
- Новая стойка меню для Самокиша
- Серия визуалов летней посадки для Тургенева
