# POINT-samokisha

## Назначение

`POINT-samokisha` — operational hub для точки `ул. Самокиша, 5Б`.

Это не замена всему `knowledge-base`, а отдельный point-level source-of-truth:
- для статуса точки;
- для локального контекста;
- для навигации по документам, подрядчикам и согласованиям;
- для перекрёстных ссылок на общие документы `VK-offee`.

- [PROJECT-STATUS.md](/Users/alexander/Github/VK-offee/POINT-samokisha/PROJECT-STATUS.md) — активные задачи и ближайшие решения
- [DOCUMENTS-INDEX.md](/Users/alexander/Github/VK-offee/POINT-samokisha/DOCUMENTS-INDEX.md) — каталог документов и перекрёстные ссылки

## Принцип структуры

Локальные point-level решения и статусы живут в `POINT-samokisha`.

Общие документы сети, юридические документы ИП, общие vendor-карточки и Pack work products остаются в своих доменных каталогах и привязываются сюда ссылками.
