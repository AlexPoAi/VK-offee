# F1: Контекст (Suprasystem + Предприниматель)

> **Система:** Suprasystem (Надсистема)
> **Роль:** Предприниматель
> **Вопрос:** Зачем? В какой контекст встраивается кофейня?

---

## 🎯 Назначение

**Целевая система:** Сеть кофеен «Вкусный Кофе»

**Надсистема:** Жизнь клиента, городская инфраструктура, рынок общепита

**Зачем существует:**
- Предоставить клиентам качественный кофе и еду
- Создать комфортное пространство для работы/встреч
- Обеспечить быстрое обслуживание

---

## 🌍 Рыночный контекст

### Целевая аудитория
- **Основная:** Офисные работники 25-45 лет
- **Вторичная:** Студенты, фрилансеры
- **Потребность:** Быстрый качественный кофе + место для работы

### Конкуренты
- Сетевые кофейни (Starbucks, Coffee House)
- Локальные независимые кофейни
- Фастфуд с кофе (McDonald's)

### Позиционирование
- **Качество:** Specialty coffee (не масс-маркет)
- **Цена:** Средний+ сегмент
- **Сервис:** Быстро + дружелюбно
- **Атмосфера:** Уютно + для работы

---

## 💰 Бизнес-модель

### Источники дохода
1. Кофе и напитки (60%)
2. Еда (боулы, выпечка) (30%)
3. Розничная продажа зерна (10%)

### Ключевые метрики
- Средний чек: 350-450₽
- Проходимость: 200-300 человек/день
- Конверсия: 70%
- Повторные визиты: 40%

---

## 🎯 Стратегические цели

### Краткосрочные (2026)
- Стабилизировать операции в 3 точках
- Внедрить систему управления знаниями
- Увеличить средний чек на 15%

### Среднесрочные (2027)
- Открыть 2 новые точки
- Запустить программу лояльности
- Автоматизировать отчётность

### Долгосрочные (2028+)
- Сеть из 10+ точек
- Франшиза
- Собственная обжарка
