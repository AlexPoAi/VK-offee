# Saby Presto API Integration

## Обзор

Интеграция с Saby Presto API для получения актуальных данных кофейни VK-offee.

## Учётные данные

Учётные данные хранятся в файле `.env` (не коммитится в git).
См. `.env.example` для примера структуры.

## Документация API

- Главная: https://saby.ru/help/integration/api
- Авторизация: https://saby.ru/help/integration/api/auth
- API Retail (товары): https://saby.ru/help/integration/api/app_sale
- API Presto: https://saby.ru/help/integration/api/app_presto

## Технические детали

- **Протокол:** OAuth 2.0
- **Формат:** JSON-RPC
- **Транспорт:** HTTPS
- **Кодировка:** UTF-8

## Типы авторизации

1. **Интерактивная** — через кнопку/ссылку в интерфейсе
2. **Сервисная** — фоновая, без участия пользователя

## Доступные данные

- Меню (блюда, напитки)
- Цены на позиции
- Остатки ингредиентов
- Статистика продаж
- Данные о сотрудниках

## Шаги интеграции

1. Зарегистрировать приложение в личном кабинете Saby
2. Получить OAuth токен
3. Настроить доступ к разделам (меню, цены, остатки)
4. Реализовать JSON-RPC клиент
5. Интегрировать в RAG-бот
