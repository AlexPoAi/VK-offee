# Инструкция по регистрации приложения в Saby

## Шаг 1: Вход в личный кабинет

1. Перейти на https://online.sbis.ru
2. Войти с учётными данными из `.env` файла

## Шаг 2: Регистрация приложения

1. В личном кабинете найти раздел "Интеграция" или "API"
2. Выбрать "Зарегистрировать приложение" или "Создать приложение"
3. Заполнить форму:
   - **Название приложения:** VK-offee RAG Bot
   - **Описание:** Интеграция с RAG-ботом для получения актуальных данных кофейни
   - **Тип авторизации:** Сервисная (фоновая)
   - **Разделы доступа:**
     - ✅ Меню (Presto)
     - ✅ Цены (Retail)
     - ✅ Остатки (Inventory)
     - ✅ Продажи (Presto)
   - **Ответственные лица:** указать контакты
   - **Внешние домены:** localhost (для тестирования)

## Шаг 3: Получение credentials

После регистрации приложения вы получите:
- **Client ID** — идентификатор приложения
- **Client Secret** — секретный ключ

## Шаг 4: Настройка .env файла

1. Скопировать `.env.example` в `.env`:
   ```bash
   cp .env.example .env
   ```

2. Заполнить полученные credentials:
   ```
   SABY_CLIENT_ID=полученный_client_id
   SABY_CLIENT_SECRET=полученный_client_secret
   ```

## Шаг 5: Установка зависимостей

```bash
cd /Users/alexander/Github/VK-offee/saby-integration
pip install -r requirements.txt
```

## Шаг 6: Тестирование подключения

```bash
python saby_client.py
```

Ожидаемый результат:


## Возможные проблемы

### Ошибка 401 Unauthorized
- Проверить правильность Client ID и Client Secret
- Убедиться что приложение активировано в личном кабинете

### Ошибка 403 Forbidden
- Проверить что у приложения есть доступ к нужным разделам
- Добавить необходимые разрешения в настройках приложения

### Ошибка подключения
- Проверить интернет-соединение
- Убедиться что API Saby доступен (https://api.sbis.ru)
