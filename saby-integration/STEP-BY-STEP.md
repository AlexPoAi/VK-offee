## Шаг 7: Первый API запрос

После получения токена можно делать запросы к API:

```python
import requests

token = open('.token').read()

headers = {
    'X-SBISAccessToken': token,
    'Content-Type': 'application/json'
}

# Пример: получить список клиентов
response = requests.post(
    'https://online.sbis.ru/service/',
    json={
        "jsonrpc": "2.0",
        "method": "CRM.GetClients",  # пример метода
        "params": {},
        "id": 1
    },
    headers=headers
)

print(response.json())
```

## Возможные проблемы

### Не могу найти раздел "Приложения"
- Проверь права доступа (нужны права администратора)
- Попробуй поиск по сайту: "API" или "интеграция"
- Обратись в поддержку: support@sbis.ru

### Ошибка 401 при авторизации
- Проверь правильность app_client_id, app_secret, secret_key
- Убедись что приложение активировано

### Ошибка 403 при API запросах
- Проверь что у приложения есть доступ к нужным разделам
- Добавь необходимые разрешения в настройках приложения
