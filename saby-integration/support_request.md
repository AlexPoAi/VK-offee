Здравствуйте!

Мы используем Saby Presto для учёта в сети кофеен "Вкусный Кофе" (3 точки).

## Задача

Необходимо автоматизировать получение данных из системы для интеграции с внутренним RAG-ботом (помощник для сотрудников).

## Требуемые данные

1. **Товары и цены** (номенклатура, актуальное меню)
2. **Документы продаж** (чеки, Z-отчёты, выручка за период)
3. **Накладные** (приход товара, списания)
4. **Остатки товаров** (текущие остатки на складе)

## Вопросы

1. **Доступен ли REST API или JSON-RPC API** для программного доступа к данным нашей организации?

2. **Требуется ли регистрация приложения?** Если да:
   - Где в личном кабинете это сделать?
   - Какие данные нужны для регистрации?
   - Как получить client_id и client_secret?

3. **Какой метод авторизации используется?** (OAuth 2.0, API ключи, токены)

4. **Есть ли документация по API** с примерами запросов для:
   - Получения списка товаров
   - Получения документов продаж
   - Получения остатков

5. **Включён ли API доступ в наш тариф?** Если нет — какой тариф требуется?

6. **Есть ли альтернативные способы автоматизации?**
   - Автоматическая выгрузка отчётов (CSV/Excel) на FTP/email
   - Webhook уведомления о новых документах
   - Интеграция через сторонние сервисы
